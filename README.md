# CyberSec Flashcards

Static study website for the UvA BITM course `Cybersecurity and IT/AI auditing`.

Live site: https://jonathanvdheuvel.github.io/cybersec-mila/

The published site is a GitHub Pages website built from the local project output. It includes the study dashboard plus embedded access to the lecture slides, reader, and tutorial PDFs. This repo intentionally tracks only the publishable website files, not the extracted course materials or working files used to build it.
