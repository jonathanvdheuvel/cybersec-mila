# CyberSec Flashcards

Static study website for the UvA BITM course `Cybersecurity Foundations I`.

Live site: https://jonathanvdheuvel.github.io/cybersec-mila/

The published site is a single-file GitHub Pages website built from the local project output. This repo intentionally tracks only the publishable website files, not the extracted course materials or working files used to build it.
